# Bitbucket

After connecting your Bitbucket account you can use its repositories as a build source for applications deployed via Wodby.

## Build templates

Bitbucket integrations can use Bitbucket repositories as app build sources. Automated import of public build templates
into new Bitbucket repositories is not supported yet.

To start from a build template with Bitbucket, create or copy the repository manually in Bitbucket, then select that
repository as the build source in Wodby.
