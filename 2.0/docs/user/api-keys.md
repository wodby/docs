# API keys

Create API keys from `User settings > API keys`.

These keys are intended for automation with the Wodby API and CLI.

## When creating a key

You choose:

- organization
- description
- expiration: `No expiration`, `1 month`, `3 months`, or `6 months`

Each key belongs to one organization. If you automate against multiple organizations, create a separate key for each one.

## After creation

- The token value is shown only once.
- Store it securely before leaving the page.
- The dashboard later shows metadata such as creation time, last use, and expiry.

## Managing existing keys

From the list page you can:

- review description
- review creation time
- review last-use time
- review expiry
- delete the key
