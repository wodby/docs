Manual /wodby stack service resources

wodby stack service resources set

Set stack service resources

Usage

wodby stack service resources set SERVICE_ID [flags]

Options

Flag Description Default
--container string Container name -
--data string JSON request body -
-f, --file string Path to JSON request body -
-h, --help help for set false
--limit-cpu int CPU limit 0
--limit-mem int Memory limit 0
--request-cpu int Requested CPU 0
--request-mem int Requested memory 0
--workload string Workload name -

Options inherited from parent commands

Flag Description Default
--access-token string Access token -
--api-base-url string Public REST API base URL https://apiv2.wodby.com/v1
--api-key string API key -
--ci-config-path string Path to CI config /tmp/.wodby-ci.json
-o, --output string Output format: table, vertical, or json table
--verbose Verbose output false
