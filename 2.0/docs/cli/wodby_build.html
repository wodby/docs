wodby build

Manage app builds

Commands

wodby build deploy Deploy build wodby build get Get build wodby build list List builds wodby build registry-login Log Docker in to a build registry wodby build void Void build images

Usage

wodby build [flags]

Options

Flag Description Default
-h, --help help for build false
-o, --output string Output format: table or json table

Options inherited from parent commands

Flag Description Default
--access-token string Access token -
--api-base-url string Public REST API base URL https://apiv2.wodby.com/v1
--api-key string API key -
--ci-config-path string Path to CI config /tmp/.wodby-ci.json
--verbose Verbose output false
