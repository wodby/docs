# Demo Kubernetes Cluster

Demo is the temporary Wodby Cloud option for testing.

To use it:

1. Start creating a new application
2. In Step 2 select `Wodby Cloud`
3. Turn on the `Demo` switch

Wodby will create a temporary cluster for the application. The demo cluster and all deployed applications are deleted automatically after 24 hours.

Demo is intended for evaluation, testing, and short-lived preview environments.
