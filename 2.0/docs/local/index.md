# Local development

We provide a few open-source projects hosted on GitHub that allow you to spin up local development environment for Drupal, WordPress, PHP, Ruby, Python and Laravel projects. All projects are based on Docker and docker-compose and are available for Linux, macOS and Windows.

- [Drupal](local/drupal.md)
- [Docker4WordPress](local/wordpress.md)
- [Docker4PHP](local/php.md)
- [Docker4Ruby](local/ruby.md)
- [Docker4Python](local/python.md)
- [Docker4Laravel](https://github.com/wodby/docker4laravel)
